## 一、P0 接口基线

### 1. 鉴权

- `POST /auth/send-code`
- `POST /auth/login`
- `POST /auth/refresh`
- `POST /auth/logout`
- `GET /auth/me`

### 2. 用户

- `GET /mini/user/info?id=:id`
- `POST /mini/user/update`
- `GET /mini/user/points`

### 3. 评论

- `GET /mini/comment?topicId=:id`
- `PUT /mini/comment`
- `POST /mini/comment/like`
- `DELETE /mini/comment?commentId=:id`
- `POST /mini/topic/recommend/accept`
- `POST /mini/topic/recommend/feedback`

### 4. 上传

- `POST /upload`
- `POST /upload/image`
- `POST /upload/avatar`
- `POST /upload/background`

### 5. 标签 / 虚拟权益 / 任务 / 装备库（P1 已完成第一版）

- `GET /mini/tag/usable`
- `GET /mini/tag/used`
- `POST /mini/tag/used`
- `GET /mini/tag/points`
- `POST /mini/tag/redeem`
- `GET /mini/goods?type=0`
- `POST /mini/goods`
- `GET /mini/taskFeat`
- `GET /mini/taskFeat/unfinish`
- `POST /mini/taskFeat`
- `GET /mini/gear/brands?type=reels|rods|lures`
- `GET /mini/gear/list`
- `GET /mini/gear/detail`

## 四、P0 联调结论

- 鉴权第一版：已形成 `send-code -> login -> me -> refresh -> logout` 闭环
- 用户第一版：当前用户资料读取、编辑资料保存、积分读取已接到独立后台
- 评论第一版：评论列表、发表评论、删除评论、评论点赞已切到独立后台
- 上传第一版：帖子图片、头像、背景图上传已切到独立后台本地上传目录
- 标签第一版：`my-tags / tag-manager` 已切到独立后台
- 虚拟权益商城第一版：`points-exchange` 的 `type=0` 页签已切到独立后台
- 任务第一版：积分页任务/成就已切到独立后台

## 五、当前遗留

- 评论点赞在“登录切换后再次进入详情页”的用户态一致性已补服务端唯一约束与显式聚合，仍建议继续做一轮真机联调确认
- 任务模块当前仍是最小可用第一版，后续还需要补更完整的任务规则、自动记账与后台配置能力
- 装备库当前已切到 PostgreSQL 优先读取，但维护源仍是 `rate/excel`
- 装备库搜索 / 推荐 / 更细筛选字段仍需继续打磨
