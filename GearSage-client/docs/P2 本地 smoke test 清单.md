# P2 本地 smoke test 清单

更新时间：2026-03-26

## 一、适用范围

本清单用于 P2 阶段本地环境固定回归，默认只针对：

- 小程序本地联调
- `GearSage-api` 本地服务
- `http://127.0.0.1:3001`

本清单不包含：

- 外网联调
- 备案问题
- 生产环境部署验收

## 四、可执行检查清单

### 4.1 登录

最短路径：

- 登录页发送验证码
- 手机号验证码登录
- 登录成功后回读 `GET /auth/me`

关键接口：

- `POST /auth/send-code`
- `POST /auth/login`
- `GET /auth/me`

通过标准：

- 能拿到 `token / refreshToken`
- `auth/me` 能返回当前用户

### 4.2 首页帖子流

最短路径：

- 打开首页
- 列表正常返回帖子

关键接口：

- `GET /mini/topic/all`

通过标准：

- 返回数组
- 至少能打开其中一条进入详情

### 4.3 帖子详情

最短路径：

- 从首页点开一条帖子
- 核对详情正文、图片、点赞态

关键接口：

- `GET /mini/topic?topicId=:id`

通过标准：

- 帖子 ID 正确
- `isLike / images / title / content` 可正常返回

### 4.4 发布与上传

最短路径：

- 上传 1 张发帖图
- 上传头像
- 上传背景图
- 发一条测试帖子

关键接口：

- `POST /upload/image`
- `POST /upload/avatar`
- `POST /upload/background`
- `POST /mini/topic`

通过标准：

- 上传返回本地 URL
- 发布成功返回 `id / status / publishTime`

### 4.5 草稿

最短路径：

- 先保存草稿
- 再读取最新草稿
- 再从草稿发布

关键接口：

- `PUT /mini/topic`
- `GET /mini/topic/tmp`
- `POST /mini/topic`

通过标准：

- 草稿 `status = 0`
- 发布后 `status = 2`

### 4.6 个人页 / 编辑资料

最短路径：

- 进入个人页
- 打开编辑资料页
- 修改昵称 / 简介 / 头像 / 背景图
- 保存后回读

关键接口：

- `GET /auth/me`
- `POST /mini/user/update`
- `POST /upload/avatar`
- `POST /upload/background`

通过标准：

- 保存成功
- 回读资料与刚保存内容一致

### 4.7 标签页

最短路径：

- 打开 `my-tags`
- 打开 `tag-manager`
- 读取已拥有标签
- 修改佩戴标签并保存

关键接口：

- `GET /mini/tag/used`
- `GET /mini/tag/usable`
- `POST /mini/tag/used`

通过标准：

- 可读取 starter tags / 已兑换标签
- 佩戴保存成功

### 4.8 积分商城

最短路径：

- 打开积分商城虚拟权益页签
- 读取商品列表
- 兑换一个未拥有标签商品
- 再回读积分与标签

关键接口：

- `GET /mini/user/points`
- `GET /mini/goods?type=0`
- `POST /mini/goods`
- `GET /mini/tag/used`

通过标准：

- 兑换成功
- 积分扣减
- 标签进入 `ownedTags`

### 4.9 任务页

最短路径：

- 打开积分任务页
- 读取未完成任务
- 领取一项任务奖励
- 再回读积分与已完成任务

关键接口：

- `GET /mini/taskFeat/unfinish`
- `POST /mini/taskFeat`
- `GET /mini/taskFeat`
- `GET /mini/user/points`

通过标准：

- 可领取任务返回成功
- 积分变化符合预期

### 4.10 装备列表 / 详情

最短路径：

- 打开装备列表
- 读取品牌
- 读取列表
- 打开装备详情

关键接口：

- `GET /mini/gear/brands`
- `GET /mini/gear/list`
- `GET /mini/gear/detail`

通过标准：

- 品牌列表非空
- 列表返回 `list / total`
- 详情返回 `variants`
