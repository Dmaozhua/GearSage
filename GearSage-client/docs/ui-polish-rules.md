# GearSage UI 微调规则

## 目标
本轮只做视觉气质优化，不做功能重构，不改信息架构。

## 关键词
专业、克制、装备资料库、低饱和、清晰、可信、轻社区感。

## 禁止
- 不使用鲜艳渐变
- 不做大面积彩色背景
- 不做幼稚插画风
- 不随意新增页面结构
- 不改变已有接口和业务逻辑
- 不把小程序改成网页 Landing Page 风格

## 允许优化
- 字号层级
- 卡片间距
- 卡片圆角
- 阴影克制化
- 分割线
- 信息密度
- 搜索框
- 筛选栏
- 标签样式
- 空状态
- 按钮主次层级
- 装备参数展示节奏

## 色彩方向
- 主色：深灰 / 炭黑 / 暗橄榄 / 卡其灰
- 背景：浅灰白
- 强调色：少量低饱和绿色或金棕色
- 避免高饱和蓝、红、紫、橙
