# 自定义底部导航栏使用说明

## 概述

本项目已成功实现了具有动态动画效果的自定义底部导航栏，替换了原有的标准tabBar。新导航栏具有以下特点：

- ✨ **动态SVG动画效果**：点击时触发流畅的矩形上升动画
- 🎯 **弹性交互反馈**：触觉反馈和视觉动画相结合
- 🌙 **夜间模式适配**：自动适配深色主题
- 📱 **响应式设计**：适配不同屏幕尺寸
- 🔄 **流畅的状态切换**：平滑的选中状态过渡

## 文件结构

```
custom-tab-bar/
```

## 实现原理

### 1. 配置启用

在 `app.json` 中启用自定义tabBar：

```json
{
  "tabBar": {
    "custom": true,
    "list": [
      {
        "pagePath": "pages/index/index",
        "text": "首页"
      },
      {
        "pagePath": "pages/points/points",
        "text": "积分"
      },
      {
        "pagePath": "pages/profile/profile",
        "text": "我的"
      }
    ]
  }
}
```

### 2. 组件结构

#### WXML模板
- 使用flex布局创建导航栏容器
- 每个tab项包含图标、文字标签和SVG动画背景
- 16个矩形元素组成动画效果

#### WXSS样式
- 深色主题设计（#1a2030背景色）
- 圆角胶囊形状（border-radius: 100rpx）
- CSS动画实现矩形上升效果
- 混合模式创建颜色叠加效果

#### JS逻辑
- 自动检测当前页面并更新选中状态
- 触觉反馈增强用户体验
- 页面跳转和状态同步

### 3. 动画效果

#### 矩形动画
```css
@keyframes slideUp {
  0% {
    transform: translateY(100%);
  }
  100% {
    transform: translateY(0%);
  }
}
```

#### 颜色系统
- **粉色矩形**：#f52cb9（主色调）
- **蓝色矩形**：#05eafa（叠加色，使用color-dodge混合模式）
- **背景色**：#1a2030（深色主题）
- **边框色**：#252e45（微妙边框）

## 页面集成

每个tabBar页面需要在 `onShow` 生命周期中更新选中状态：

### 首页 (index.js)
```javascript
onShow() {
  // 更新自定义tabBar选中状态
  if (typeof this.getTabBar === 'function' && this.getTabBar()) {
    this.getTabBar().setData({
      selected: 0
    });
  }
}
```

### 积分页 (points.js)
```javascript
onShow() {
  // 更新自定义tabBar选中状态
  if (typeof this.getTabBar === 'function' && this.getTabBar()) {
    this.getTabBar().setData({
      selected: 1
    });
  }
}
```

### 个人页 (profile.js)
```javascript
onShow() {
  // 更新自定义tabBar选中状态
  if (typeof this.getTabBar === 'function' && this.getTabBar()) {
    this.getTabBar().setData({
      selected: 2
    });
  }
}
```

## 特性详解

### 1. 安全区域适配

```javascript
attached() {
  // 获取安全区域信息
  const systemInfo = wx.getSystemInfoSync();
  this.setData({
    safeAreaBottom: systemInfo.safeArea ? 
      systemInfo.screenHeight - systemInfo.safeArea.bottom : 0
  });
}
```

### 2. 触觉反馈

```javascript
triggerAnimation(index) {
  // 添加触觉反馈
  wx.vibrateShort({
    type: 'light'
  });
}
```

### 3. 响应式设计


## 自定义配置

### 修改导航项

在 `custom-tab-bar/index.js` 中修改 `list` 数组：

```javascript
list: [
  {
    pagePath: "/pages/index/index",
    iconPath: "/images/icons/home.png",
    selectedIconPath: "/images/icons/home-active.png",
    text: "首页"
  },
  // 添加更多导航项...
]
```

### 修改颜色主题

在 `custom-tab-bar/index.wxss` 中修改颜色变量：

```css
.tab-nav {
  background: #your-background-color;
  border: 2rpx solid #your-border-color;
}

.pink-rect {
  background: #your-primary-color;
}

.blue-rect {
  background: #your-secondary-color;
}
```

### 调整动画效果

修改动画持续时间和缓动函数：

```css
.tab-item.active .pink-rect {
  animation: slideUp 1.8s cubic-bezier(0.68, -0.55, 0.265, 1.55) forwards;
}
```

## 故障排除

### 常见问题

1. **导航栏不显示**
   - 检查 `app.json` 中是否正确设置 `"custom": true`
   - 确认 `custom-tab-bar` 文件夹在项目根目录

2. **选中状态不更新**
   - 检查页面的 `onShow` 方法是否正确调用 `getTabBar().setData()`
   - 确认索引值是否正确

3. **动画效果异常**
   - 检查CSS动画是否被其他样式覆盖
   - 确认浏览器是否支持相关CSS特性

4. **安全区域适配问题**
   - 检查 `systemInfo.safeArea` 是否正确获取
   - 手动设置 `padding-bottom` 值

### 调试技巧

1. 在开发者工具中查看组件结构
2. 使用 `console.log` 输出调试信息
3. 检查CSS样式是否正确应用
4. 测试不同设备的显示效果
