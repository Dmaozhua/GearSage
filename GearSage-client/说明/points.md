*积分基础说明
*积分获取，每日任务（0点刷新，固定任务）、一次性任务、更多任务。
*积分最小增加是1，积分目前有效期是永久。
*积分只用来兑换标签，（兑换商品 没有资质），同1标签1个用户只能兑换1次，标签种类可增加。
*积分使用记录查看 选做。

*任务设计
每日任务
3分 每日签到→点击签到按钮→ 任务跳转 按钮 变成可领取状态 → 点击领取→积分增加
1分 互动点赞→ 点击去完成 → 跳转至index →给任意一个帖子点赞后（再取消赞不做考虑）→ 按钮变成可领取状态 → 点击领取→积分增加
1分 进行评论→ 点击去完成 → 跳转至index →给任意一个帖子评论后→ 按钮变成可领取状态 → 点击领取→积分增加
10分 发布帖子→ 点击去完成 → 跳转至index →发布任意板块（未来多板块）的帖子（审核通过后）→ 按钮变成可领取状态 → 点击领取→积分增加
一次性任务（跨天是0点）
5分 编辑资料→ 触发了保存接口true → 按钮变成可领取状态 → 点击领取→积分增加
（如果可以拆的话，头像是一个，资料是一个）
5分 首次登录→ 登录成后 → 按钮变成可领取状态 → 点击领取→积分增加
10分 累计登录3天→ 累计登录3天→ 按钮变成可领取状态 → 点击领取→积分增加
10分 累计登录5天→ 累计登录5天→ 按钮变成可领取状态 → 点击领取→积分增加
20分 分享1次
其他任务
20分 连续登录7天→ 连续登录7天→
累计登录3天→ 按钮变成可领取状态 → 点击领取→积分增加
30分 邀请好友注册
